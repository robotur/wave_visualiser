# Piano Wave Visualizer

A Python GUI application that visualizes sound waves generated by piano notes using triangle waves.

## Requirements

- Python 3.6+
- NumPy
- Matplotlib
- Tkinter (usually included with Python)

## Installation

```bash
pip install numpy matplotlib
```

## Usage

```bash
python main.py
```

### Controls

- **Left Click**: Play a note and display its wave
- **Right Click**: Hold/release a note (shaded when held)
- **Octave Input**: Change octave range (0-8), click "Update" or press Enter

### Features

- 2-octave piano keyboard (14 white keys, 10 black keys)
- Real-time triangle wave visualization (zoomed in at 0.05 seconds)
- Chord support: Hold multiple notes to visualize combined waves
- Adjustable octave range with visual feedback

## Technical Details

- Sampling frequency: 44,100 Hz
- Wave type: Continuous triangle waves
- Default octave: 4 (middle C octave)
- Frequency calculation: Based on A4 = 440 Hz standard tuning
